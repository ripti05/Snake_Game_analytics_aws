# 🐍 Snake Game with AWS CI/CD

This project is a web-based Snake Game with a Node.js backend, deployed using a complete CI/CD pipeline on AWS.

The application is connected to GitHub, and every code change automatically triggers the pipeline. AWS CodePipeline manages the workflow, CodeBuild installs dependencies, and CodeDeploy deploys the application to an EC2 instance where the game runs.

The project demonstrates how to automate build and deployment processes using cloud services, reducing manual effort and improving efficiency.

## Tech Used
- HTML, CSS, JavaScript (Frontend)
- Node.js, Express (Backend)
- AWS CodePipeline, CodeBuild, CodeDeploy
- Amazon EC2

## Key Idea
Automated deployment of a web application using AWS CI/CD services.
